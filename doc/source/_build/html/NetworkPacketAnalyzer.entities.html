NetworkPacketAnalyzer.entities.Flow module¶

Flow¶

定义流的实体。

每一个流分为前后两个方向。当一个包到来时，会根据它的头部信息判断它属于前向包还是后向包。 每一个属于该流的数据包被加入该流时，相应的统计量会进行更新。
